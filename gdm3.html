gdm3 Lockscreen Background

Change the lock screen background on Gnome/ gdm3 of Ubuntu or Pop OS 20.04 and 20.10 and other Ubuntu based Distros.

Web

For now this project supports Ubuntu/Pop-Os-20.04 and 20.10 version! New support for Arch linux Gnome environment. New support for Fedora 31 onwards. This script won't work with any previous versions of Ubuntu/Pop-Os because they comes with older version of gdm rather than gdm3.36. It also won't work if your system is set to a custom gdm3 theme. You will have to reset to the default configuration of gdm3 before using the script. This script is for default Ubuntu theme (Yaru). This tool was made specifically to work with Ubuntu or Pop OS 20.04 and 20.10 as it now bundles all configuration files inside a .gresource file.

  • Change login wallpaper
  • Wallpapers located in /usr/share/Wallpapers
  • Enable / disable user for login as changes takes place for only one user.
  • Proven to work gnome-shell 3.36 onwards as it has support for it.
  • Run the script with root privileges.
